## About

A personal portfolio that showcases my journey as a developer — from front-end fundamentals to full-stack work with .NET backends, multi-tenant SaaS, and QA experience. Features a curated project showcase, interactive learning timeline, dark mode, and bilingual support (EN / FR).

## Features

- **Bilingual** — full English and French translations via i18n
- **Dark mode** — persisted across sessions via localStorage
- **Featured projects** — live demo and GitHub links on every card
- **Learning timeline** — chronological career and project history
- **Responsive** — mobile-first layout with Tailwind CSS
